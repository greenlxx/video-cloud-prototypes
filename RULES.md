# 视频云平台 — 原型导航首页规则

## 一、版本规则

### 版本分类
| 类型 | 命名格式 | 示例 | 说明 |
|------|---------|------|------|
| **基线版本** | `vYYYYMM` | v202606 | 平台自身迭代，按月发版 |
| **市场化项目** | `项目名-vYYYYMM` | 龙江市监-v202606 | 客户定制项目，独立跟踪 |

### 版本切换
- 导航首页顶部有「版本选择器」，下拉切换版本
- 每个版本独立维护一套页面配置
- 默认显示最新版本

---

## 二、菜单层级规则

### 页面结构
```
一级菜单（功能域）
  ├── 二级菜单 1（模块A）
  │    ├── 功能1（全新/改造/保留）
  │    └── 功能2（全新/改造/保留）
  ├── 二级菜单 2（模块B）
  └── 二级菜单 3（模块C）
```

### 一级菜单（功能域）颜色规范
| 一级菜单 | 色值 | 用途 |
|---------|------|------|
| 平台概览 | `#165DFF` | 首页、概览 |
| 数据治理 | `#722ED1` | 治理相关 |
| 智能运维 | `#13C2C2` | 运维相关 |
| 轨迹查询 | `#1677FF` | 轨迹跟踪 |
| 布控管理 | `#FA8C16` | 布控相关 |
| 设备管理 | `#52C41A` | 设备 |
| 报警中心 | `#FF4D4F` | 告警 |
| 系统管理 | `#86909C` | 设置 |

## 三、页面状态标签规则

| 标签 | 颜色 | 含义 | 卡片样式 | 边界条件 |
|------|------|------|---------|---------|
| **🔴 全新** | `#FF4D4F` | 本次迭代新建 | 左侧 3px 红条 | 之前版本不存在的页面 |
| **🟠 改造** | `#FAAD14` | 现有页面优化 | 左侧 3px 橙条 | 功能、交互、UI 有实质性变更 |
| **🟢 保留** | `#52C41A` | 现有功能参考 | 左侧 3px 绿条 | 页面存在但本次未改动 |
| **⚪ 占位** | `rgba(0,0,0,0.25)` | 后续迭代 | 左侧 3px 灰条 | 已规划但本期不开发 |

---

## 四、统计栏规则

底部统计自动计算：
- **全新页面** — 标签为「全新」的数量
- **改造模块** — 标签为「改造」的数量
- **原型文件** — 所有原型文件总数
- **可导航页面** — 已配置 URL 且可点击的页面数

## 六、导航首页配置规则

```javascript
// index.html 里的 PAGES 配置示例
const VERSION = {
  name: "v202606",           // 版本号
  type: "baseline",          // baseline | project
  project: "",               // 市场化项目时才填
  date: "2026-06",           // 所属月份
  label: "6月基线版本"       // 显示名称
};

const PAGES = [
  {
    menu1: "数据治理",        // 一级菜单
    menu2: "点位智能标注",     // 二级菜单
    icon: "🤖",
    iconBg: "#f9f0ff",
    title: "点位智能标注",
    desc: "Epic-G · 业务标签智能标注",
    type: "new",              // new | change | keep | stub
    pages: "list.html + detail.html",
    url: "v202606/dg-annotation/list.html",
    files: [                  // 此需求的所有原型文件
      { name: "标注列表", url: "v202606/dg-annotation/list.html" },
      { name: "标注详情", url: "v202606/dg-annotation/detail.html" }
    ]
  }
];
```
